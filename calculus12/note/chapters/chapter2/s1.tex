\subsection{The Derivatives of Polynomial Functions}
\subsubsection{What is a power?}
To start off, we need to define the power function.
$$x^a, x \in \R, a \in \R$$
At here, $x$ is the base, and $a$ is the exponent. 
Base and exponent together is called "Power". \\
$$x^a$$ should be read as:
\begin{center}
    "x to a"
\end{center}

\subsubsection{Power Rules}
If $$f(x) = x^n$$
the derivative of $f(x)$ is
$$f'(x) = nx^{n - 1}$$

In words:
\begin{center}
    "To find the derivative of a power function, take the exponent from the power function and use it as the coefficient of the derivative.
    Then reduce the exponent value for the derivative by 1 from the original function."
\end{center}

\begin{remark}
    Do not use "Power Rule" for exponent function ($a^x$), such as $2^x$
\end{remark}

\subsubsection{Derivative Rules}
\begin{center}
    If $f(x) = c$, $f'(x) = 0$\\

    If $f(x) = x^n, f'(x) = nx^{n - 1}$\\

    If $f(x) = cg(x), f'(x) = c * g'(x)$\\

    If $h(x) = f(x) + g(x), h'(x) = f'(x) + g'(x)$\\

    If $h(x) = f(x) - g(x), h'(x) = f'(x) - g'(x)$\\
\end{center}
