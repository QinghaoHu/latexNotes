\subsection{Sketching Rational Function}
This chapter is pretty similar to the previous notes. 

In Addition to the previous note, there are some rules you need to be careful about:

\begin{definition}
    [The Product Rule]
    Given function $f(x) = g(x)h(x)$
    \begin{align*}
        f'(x) = g'(x)h(x) + g(x)h'(x)
    \end{align*}
\end{definition}

\begin{definition}
    [The Quotient Rule]
    Given function $f(x) = \frac{g(x)}{h(x)}$
    \begin{align*}
        f'(x) = \frac{
            g'(x)h(x) - g(x)h'(x)
        }{
            h(x)^2
        }
    \end{align*}
\end{definition}
