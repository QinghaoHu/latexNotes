\subsection{Curve Sketching}
In order to graph the curve, we have developed three strategy in the previous lessons. 

Given:
\begin{align*}
    f: \R \to \R
\end{align*}

\subsubsection{Original Function $f(x)$}
We can apply line tests on the original function to find the property. 
\begin{itemize}
    \item If $f(x) < 0$, the original function is negative
    \item If $f(x) > 0$, the original function is positive 
\end{itemize}

\subsubsection{First Derivative $f'(x)$}
We can apply line tests on the first derivative to find the property. 
\begin{itemize}
    \item If $f'(x) < 0$, the original function is decreasing
    \item If $f'(x) > 0$, the original function is increasing
\end{itemize}

\subsubsection{Second Derivative $f''(x)$}
We can apply line tests on the second derivative to find the property. 
\begin{itemize}
    \item If $f''(x) < 0$, the original function is concave down
    \item If $f''(x) > 0$, the original function is concave up 
\end{itemize}
