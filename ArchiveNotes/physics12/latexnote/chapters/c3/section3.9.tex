\section{HEAD-ON Collisions}
\subsection{Equations}
Under this section, I will derive equations that are useful for solving the final velocity of each object when you are given the initial velocity in a \text{HEAD-ON collision} question.

\begin{redblock}
    The equation for $\vec{v_A}'$
   \begin{align*}
    \vec{v_A}' &= \frac{m_A - m_B}{m_A + m_B}\vec{v_A}
   \end{align*}
   The equation for $\vec{v_B}'$
   \begin{align*}
    \vec{v_B}' &= \frac{2m_A}{m_A + m_B}\vec{v_A}
   \end{align*}
\end{redblock}

For questions in which $\vec{v_B}$ is not zero, we need to change the frame of reference to $\vec{v_B}$ in order to solve the question, then change the frame of reference back afterward.
