\section{Relative Velocities in Two Dimensions}
\subsection{Recall}
\begin{enumerate}
    \item All velocities are defined relative to a frame of reference that we consider to be \textcolor{red}{motionless}.
    \item A frame of reference is a \textcolor{red}{perspective} from which we observe or measure the motion of objects.
\end{enumerate}
\subsection{Definition}
We sometimes refer to something that another object can travel through as a \textit{medium}. We also refer to the object moving through the medium as \textit{object (O)}, and the ground uses the symbol \textit{G}.

Therefore the equation can be written as:
\[
\vec{v_{OG}} = \vec{v_{OM}} + \vec{v_{MG}}
\]
