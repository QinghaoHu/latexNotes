\section{An introduction to problem solving techniques}
\subsection{Steps in problem solving}
\subsubsection{Prerequisites for problem solving}
\begin{enumerate}
    \item Number sense and number skills
    \item Pattern recognition
    \item Patience, humility, ability to learn from mistakes, perseverance
\end{enumerate}

\subsubsection{Preliminary steps}
\begin{enumerate}
    \item Read the problem carefully and understand each word in the statement.
    \item Understand what is given and what is asked for.
    \item Search our memory for relevant and important facts that relate to the things that we're given and the things that we want to find.
    \item Search our memory for similar problems that we have previously solved.
\end{enumerate}

\subsubsection{Solving the problem}

\subsubsection{Final steps}
\begin{enumerate}
    \item Did I get an answer or solution?
    \item Does my answer actually answer this question?
    \item Does my answer make sense?
    \item Make up your own problem from time to time?
\end{enumerate}
